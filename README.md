# A cookbook for Lodash FP

Common patterns for using Lodash/FP

## What is this?

FP-Cookbook is a collection of examples of how to use Lodash/FP.

## Example

```js
const library = {
  name: "Branch 1",
  address: "Station Road",
  bookcount: 3000
};

const name = get("name", library);
// name: Branch 1
```
